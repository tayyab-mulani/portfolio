# Portfolio Website

Personal portfolio website built with React 18 + Vite, showcasing projects, skills, experience, and a contact form.

## Pages

- **Home** — Hero section with introduction and resume download
- **About** — Personal info, stats, skills with progress bars, and resume timeline
- **Projects** — Project cards with modals (description, tech stack, links)
- **Contact** — Contact form powered by EmailJS + social links
